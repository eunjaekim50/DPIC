# check-in — 매주 방향 점검

미카가 만든 첫 스킬. 매주 "내 방향이 어디로 움직였나"를 되묻고, 답을 journey.md에 기록으로 남긴다.

## 절차

1. **읽는다**
   - `journey.md`의 "나의 선언"
   - `journey.md`의 "나의 진행 상태" 표 (가장 최근 주차 행)
   - `outputs/` 아래 이번 주에 새로 생기거나 수정된 파일

2. **요약한다** (한 문단, 대필 금지 — 사실만)
   - 이번 주 outputs/에 무엇이 쌓였는지
   - 나의 선언과 비교했을 때 방향이 그대로인지, 좁혀졌는지, 다른 쪽으로 옮겨갔는지 관찰만 짧게 제시

3. **묻는다** (AskUserQuestion 사용, 선택지 + 자유 입력 허용)
   - "지난주와 비교해 내 방향이 어떻게 움직였나요?" — 예: 더 좁혀졌다 / 그대로다 / 다른 쪽으로 옮겨갔다 / 아직 모르겠다
   - "이번 주 강연·조사에서 내 방향에 걸린 게 있었나요?" — 있으면 한 줄로
   - "VC 말고 다른 산업에 눈이 간 순간이 있었나요?" — 있으면 한 줄로

4. **기록한다**
   - 참가자의 답변을 받아 적어 `journey.md`의 "세션 노트" 섹션에 한 줄 추가:
     `- YYYY-MM-DD 체크인: {방향 이동 답변} / 걸린 것: {있으면 한 줄, 없으면 생략} / 다른 산업: {있으면 한 줄, 없으면 생략}`
   - 대필 금지 — 참가자가 고른/말한 표현을 그대로 쓴다.
   - 파일을 고치기 전에 추가될 줄을 보여주고 확인받는다.

## 규칙

- 투자 조언·종목 추천 금지 (클럽 공통 규칙, CLAUDE.md 참조).
- 판단은 참가자가 한다 — 이 스킬은 묻고 기록만 한다.
- 매주 짧게: 전체가 3턴 이내로 끝나야 한다.
